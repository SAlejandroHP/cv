UbCubo

El proyecto PLD esta basado en todas las normas indicadas para la prevencion de lavado de dinero, se encarga de dar de alta clientes para la SOFOM y estos son validados en una BD (MySql) en la cual contiene listas negras, listas de personas bloqueadas y listas PEPS (Personas Politicamente Expuestas), una vez que se validan los clientes dentro de las listas, se pueden agregar movimientos a sus prestamos que dependientemente da la SOFOM, en caso de que algun cliente rebase la normativa arroja alertamientos.

Este programa esta realizado con el framework CodeIgniter, la base de datos esta echa en MySql y realiza peticiones a travez de AJAX-JSON Para los estilos, esta conformado por una plantilla realizada con Bootstrap.

GSI Seguridad

Cotizador

PHP (CodeIgniter) y esta conectado a una base de datos que esta en progress con peticiones JSON.
